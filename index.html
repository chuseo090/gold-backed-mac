2. Core Mechanism & Product Features

G-MAC is an advanced collateralized stablecoin platform. Our V3 logic focuses on maintaining a rigid 150% collateral ratio, setting a new standard for decentralized asset stability.

Minting (Issue G-MAC)

Users deposit MAC Token collateral to mint (issue) G-MAC stablecoin.

  • **Ratio:** 1.5 MAC : 1 G-MAC (150% Overcollateralized)
  • **Status:** V3 Audited Logic.

Redemption (Burn G-MAC)

Users burn G-MAC stablecoin to redeem the underlying MAC collateral.

  • **Logic:** Redemption uses the same 1.5:1 ratio, ensuring fair value recovery.
  • **Audit Note:** All reentrancy and security risks resolved in V3.
